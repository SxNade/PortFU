# PortFU

![](https://media.tenor.com/images/9533937aefa8dfacbee26a9f23d26c61/tenor.gif)

PortFU is a bash implementation of a port scanner

# Installation and running


            $ git clone https://github.com/SxNade/PortFU
            $ cd PortFu
            $ chmod +x portfu
            
            $ ./portfu <host-ip>
